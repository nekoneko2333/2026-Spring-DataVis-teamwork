- 读取方式（Python 示例，务必验证一次）：
    arr = np.fromfile(path, dtype='<f4')                 # little-endian float32
    # 列优先 = flat index 为 z + nz*(y + ny*x)，即 z 变化最快。两种等价写法：
    vol_zyx = arr.reshape((128, 128, 128), order="F")    # 直接得到 axes:(z, y, x)
    # 或：
    vol_xyz = arr.reshape((128, 128, 128), order="C")    # axes:(x, y, z)
    vol_zyx = np.transpose(vol_xyz, (2, 1, 0))           # 转成 (z, y, x)
  注意：必须输出 xy / xz / yz 三个方向的切片图人工核对轴向（无错位/转置/镜像），确认后把正确写法固化在代码里，不要想当然。

三、创新点（重点，决定能否拿高分）
================================================================
【创新点 A：合成莱曼-α 森林光谱（近似 proxy 模型）】（性价比最高，与作业背景强绑定）
- 在体数据中沿任意视线投射射线，光学深度 τ 由密度积分给出，透射流量 F = exp(−τ)。
- 物理限定（务必在 UI 与报告中明确写出）：当前数据仅含气体密度，缺温度、电离态、速度场，无法做严格辐射转移。
  故采用**教学/可视分析近似模型**：τ = A·∫ ρ^β ds，A、β 可交互调节；这是**密度驱动的 proxy**，不是完整 Lyman-α 辐射转移模型。
- 交互：用户在 3D 视图中拉出一条视线 → 实时生成该视线的 1D 吸收光谱。
- 价值：把三维密度结构与真实天文观测量（Lyman-α forest）直接关联，体现物理理解而非单纯美观。

【创新点 B：宇宙网形态学分类（Hessian 特征值法）】
- 对平滑后的场计算 Hessian / 潮汐张量，求 3 个特征值，按正特征值个数把体素分为 void / sheet(墙) / filament(丝) / node(节点) 四类。
- 物理限定（避免概念过度宣称）：严格 T-web 是对**引力势** Hessian 分类。
  · 若仅用密度场 → 实现 **smoothed log-density Hessian morphology proxy**，报告中如实说明是形态学近似。
  · 若额外解 Poisson 方程求得势场 φ → 再升级为更严格的 **T-web** 分类（参考 Hahn 2007、Forero-Romero 2009）。
- 比简单按密度分位数切档科学得多，叠加成 Cosmic Atlas Overlay。

【创新点 C：量化"两极分化"演化】
- 除 Gini/偏度/峰度外，计算功率谱 P(k) 或两点相关函数随时间演化，定量证明结构增长。
- 输出"密度演化指纹图"（时间×log密度 2D 热力图）作为报告核心图。

【创新点 D：Cosmic Probe 宇宙探针（演示型，强烈推荐）】
- 用户在 3D 视图中点击或拖出一条线，沿线实时显示：① 密度剖面曲线；② 对应的莱曼-α proxy 光谱。
- 当线穿过 node / filament / void 时，剖面与光谱形态明显不同，直观展示"统计↔空间结构"对应关系。
- 比单纯切片更适合答辩现场演示。
